About

// why

Most engineers use databases, servers, and runtimes as black boxes. This project exists to open them up — to understand not just how to use these systems, but how they actually work at every layer.

Every component is built from scratch in C or C++, with no external libraries where avoidable. The goal is deep understanding, not shortcuts.

// principles
01 Build it before you use it
02 Every component must talk to another
03 Measure everything — benchmarks or it didn't happen
04 Write about what you build
// stack

Primary languages are C and C++. C for low-level systems work (networking, containers), C++ for engines and data structures where RAII and templates help without hiding complexity.

// this site

This website is served by net-stack — the HTTP server built as part of this project. No nginx, no Apache. Raw TCP sockets, a hand-written HTTP parser, and a render_template engine, all in C.
